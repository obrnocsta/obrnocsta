## Descrição do Projeto
O projeto **Wild Horizons** fornece uma API que permite acesso a informações sobre habitats e espécies em extinção. Ele é projetado para conectar desenvolvedores a dados essenciais para pesquisa e conservação.

## Projetos 
### Wild Horizons API
A API Wild Horizons permite a consulta de dados de forma simples e eficiente. Com endpoints que retornam dados em formatos JSON e XML, a interface é intuitiva e oferece acesso a:
- Informações sobre espécies
- Dados sobre habitats
- Ações de conservação

### Construindo um Site Roteado
Neste projeto, você aprenderá a construir um site utilizando as melhores práticas de HTML, CSS e JavaScript. Os principais tópicos incluem:
- Estruturação de páginas usando HTML semântico
- Estilização com CSS moderno
- Interatividade com JavaScript e bibliotecas populares
